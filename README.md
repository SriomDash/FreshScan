# 🍎 FreshScan: AI Fruit Quality Inspector

**FreshScan** is an AI-powered inventory management system capable of classifying fruits as **Fresh** or **Rotten** in real-time. It uses a deep learning model (MobileNetV2) to simulate a warehouse scanning process, updating inventory counts and providing visual analytics.

---

## 🚀 Features

- **Multi-Fruit Support:** Detects Apples, Bananas, Grapes, Oranges, and Pomegranates  
- **Quality Detection:** Classifies items as *Fresh* or *Rotten* with high confidence  
- **Live Simulation:** Simulates a warehouse camera feed scanning a batch of items  
- **Real-time Inventory:** Updates inventory counts and generates instant graphs  
- **Cloud Ready:** Optimized for deployment on **Hugging Face Spaces**
